# WebStudio Project

A modern, responsive web portfolio project featuring a professional design with modal interactions, responsive navigation, and a comprehensive portfolio showcase built with SCSS and JavaScript. Simply open `index.html` or `portfolio.html` in your browser to view the fully responsive site optimized for mobile, tablet, and desktop devices.
